🎯 Fix applicati

  • Dashboard grid: repeat(auto-fit, minmax(320px, 1fr)) → 1fr su mobile.
  • NEXUS chat fullscreen: 100vw × 100dvh (dynamic viewport, evita che la tastiera iOS copra l'input).
  • Input NEXUS: position: sticky; bottom: 0 + padding-bottom: env(safe-area-inset-bottom) (no-notch iPhone).
  • Textarea font-size 16px: evita zoom automatico Safari mobile.
  • Sidebar overlay: shadow + backdrop con ::after per dimming.
  • Touch target 44×44: sidebar-link, menu-btn, nexus-fab, bottoni archive/clear (media query hover: none).
  • FAB safe-area: bottom: calc(16px + env(safe-area-inset-bottom, 0)).
  • CHRONOS/PHARO grid: 2-colonne su tablet (≤768px), 1-colonna su smartphone (≤480px).
  • Stat row: flex-wrap: wrap per evitare overflow.
  • Tabelle: overflow-x: auto + min-width 420px per scroll orizzontale.
  • Disabilita hover translate su touch device (no flicker).
  • Emoji font: fix font-family del bottone close/clear per mostrare 🗑️ correttamente.
