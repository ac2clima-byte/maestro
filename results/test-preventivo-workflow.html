✅ Implementazione completa

Creato handlers/preventivo.js — workflow sincrono chiamato dal NEXUS router (non listener async come orchestrator.js, così Alberto vede la bozza subito).

Pipeline 5 step:

  1. Parsing input: parsePreventivoInput() estrae condominio, committente, P.IVA, oggetto dal messaggio (regex + fallback da context)
  2. Arricchimento azienda: arricchisciAzienda() chiama Haiku con prompt dedicato per estrarre ragione sociale, indirizzo, PEC, SDI; cache in memo_aziende/piva_XXX
  3. Ricerca condominio CRM: cercaCondominio() scan crm_clienti + cosmina_impianti su COSMINA (garbymobile-f89ac); ritorna indirizzo + impianti
  4. Bozza JSON strutturata: generaPreventivoJson() chiama Claude Sonnet 4.6 con prompt dettagliato; ritorna JSON con intestatario, voci, totali, condizioni
  5. Salva: doc in calliope_bozze (status=da_approvare) + record in charta_preventivi

+ intercept approvazione: tryInterceptPreventivoApproval() cattura "approva"/"modifica"/"rifiuta" nel turno successivo, aggiorna bozza + accoda invio ECHO + aggiorna charta_preventivi.
