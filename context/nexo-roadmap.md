# NEXO Roadmap

## Fase 0 — LOOP (in corso)
Alberto scrive su NEXUS una richiesta → arriva a Claude Code + Claude Chat → viene risolta.

## Fase 1 — STABILIZZAZIONE (questa settimana)
- Fix token auth per Claude Code → sblocca test automatici
- Polling IRIS funzionante 24/7
- Linguaggio naturale verificato end-to-end
- Tutti i Colleghi rispondono con dati reali
- PWA installabile e responsive

## Fase 2 — FORGE (settimana prossima)
- Claude Code testa e migliora NEXO in autonomia via NEXUS
- Test automatici continui
- Loop dev-analysis → Claude Chat approva → Code implementa

## Fase 3 — WORKFLOW (2 settimane)
- Preventivo completo con PDF da GRAPH
- Guasto urgente end-to-end
- Incassi automatici
- Sollecito pagamento
- Pattern recognition

## Fase 4 — AUTONOMIA (1 mese)
- Digest mattutino WA
- WA aziendali intercettati
- Push notification
- NEXO si auto-migliora
- Conversazione vocale naturale

## Fase 5 — ESPANSIONE (2-3 mesi)
- Sistema operativo aziendale
- MEMO conosce tutto
- DELPHI previsioni
- Multiutente (Sara, tecnici)
